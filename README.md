# בתיה - אתר פורטפוליו אדריכלות

אתר פורטפוליו מקצועי לאדריכלית, בנוי ב-HTML, CSS ו-JavaScript טהור.

## מבנה הפרויקט

```
אתר בתיה/
├── index.html      # עמוד ראשי עם כל הסקשנים
├── styles.css      # עיצוב וסגנונות
├── script.js       # פונקציונליות ואינטראקטיביות
└── README.md       # קובץ זה
```

## תכונות

- **עיצוב מודרני ומינימליסטי** - עיצוב נקי עם הרבה מקום לבן
- **תמיכה מלאה ב-RTL** - מותאם לעברית וכיוון ימין לשמאל
- **רספונסיבי** - נראה מעולה על מובייל, טאבלט ומחשב
- **ניווט חלק** - גלילה חלקה בין הסקשנים
- **טופס יצירת קשר** - עם ולידציה בסיסית
- **אנימציות עדינות** - אפקטי hover וטרנזישנים

## סקשנים

1. **Hero** - עמוד בית עם הצגה ראשונית
2. **Projects** - גלריית פרויקטים עם תמונות ותיאורים
3. **About** - מידע על האדריכלית
4. **Contact** - טופס יצירת קשר ופרטי התקשרות

## שימוש

1. פתח את `index.html` בדפדפן
2. האתר מוכן לשימוש!

## התאמה אישית

### הוספת תמונות

החלף את ה-placeholders בתמונות אמיתיות:
- `hero-placeholder` - תמונה ראשית בעמוד הבית
- `project-placeholder` - תמונות פרויקטים
- `about-placeholder` - תמונה אישית

### שינוי צבעים

ערוך את המשתנים ב-`styles.css`:
```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #34495e;
    --accent-color: #e74c3c;
    /* ... */
}
```

### הוספת פרויקטים

הוסף כרטיסי פרויקט חדשים ב-HTML בתוך `.projects-grid`:
```html
<div class="project-card">
    <div class="project-image">
        <img src="path/to/image.jpg" alt="Project description">
    </div>
    <div class="project-info">
        <!-- Project details -->
    </div>
</div>
```

### חיבור טופס יצירת קשר לשרת

ערוך את הפונקציה `contactForm.addEventListener` ב-`script.js` והסר את ההערות מהקוד שמחבר לשרת.
